# Installation

- Clone project
- Run `npm install`
- Run `npm run dev`
- App wll run in `localhost:8080`

# Development

- Run `npm run dev`
- App wll run in `localhost:8080`

# Testing

- Run `npm test`
- Run `npm run test:cover`
- Open file from directory `./coverage/lcov-report/index.html` in browser
